# FRONTEND .env.local Configuration Guide

## API Configuration

# Should match your backend server

VITE_API_BASE_URL=http://localhost:5000
VITE_SOCKET_URL=http://localhost:5000

## Optional Features

VITE_ENABLE_LOGS=true # Set to false to disable console logging

## NOTE:

# - All variables must start with VITE\_ to be accessible in the browser

# - Create .env.local for local development (not committed to git)

# - Create .env.production for production builds
